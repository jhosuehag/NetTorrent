# Utorrent Android 📥

**Utorrent Android** es tu aplicación de confianza para descargar archivos grandes, películas y música directamente en tu teléfono, de forma rápida y sencilla.

## 🚀 ¿Qué puedes hacer con esta App?

- **Descargas Rápidas:** Baja archivos pesados utilizando tecnología torrent, que aprovecha al máximo tu conexión a internet.
- **Soporte Completo:** Puedes abrir enlaces automáticos (Magnet links) o archivos `.torrent` que encuentres en la web.
- **Directo a tu Carpeta:** Los archivos se guardan automáticamente en la carpeta de "Descargas" de tu teléfono para que los encuentres fácilmente.
- **Control en Tiempo Real:** Mira el progreso de tus descargas, la velocidad y cuánto falta para que terminen.

## ⭐ Características Principales

- **Fácil de Usar:** Interfaz limpia sin botones complicados. Solo agrega tu torrent y empieza a descargar.
- **Sin Interrupciones:** Funciona en segundo plano para que puedas seguir usando tu celular mientras se completan tus descargas.
- **Buscador de Nodos:** Encuentra fuentes de descarga rápidamente para que no te quedes esperando.

## 📱 ¿Cómo empezar?

1. **Abre la aplicación.**
2. **Otorga los permisos** para guardar archivos en tu carpeta de descargas (es fundamental para que la app funcione).
3. **Pega un enlace Magnet o abre un archivo .torrent.**
4. **¡Listo!** El archivo comenzará a descargarse solo.
